# Editing Content in CMS

## Basic Editing

### Opening an Entry
1. Navigate to collection
2. Find entry in list
3. Click to open editor
4. Make changes
5. Save or publish

### Field Types

#### Text Fields
- Single line text
- Multi-line text
- Rich text (markdown)

#### Media Fields
- Image upload
- File upload
- Image with alt text

#### List Fields
- Repeating items
- Ordered lists
- Nested structures

#### Date Fields
- Date picker
- Time picker
- Date/time combination

## Rich Text Editing

### Markdown Support
- **Bold**: `**text**`
- **Italic**: `*text*`
- **Links**: `[text](url)`
- **Headers**: `# H1`, `## H2`
- **Lists**: `- item` or `1. item`

### WYSIWYG Features
- Formatting toolbar
- Link insertion
- Image insertion
- List creation

## Image Management

### Uploading Images
1. Click image field
2. Choose "Upload" or "Select"
3. Select file from computer
4. Add alt text
5. Save

### Image Best Practices
- **Format**: JPG or PNG
- **Size**: Optimize before upload
- **Alt Text**: Always add descriptive alt text
- **Naming**: Use descriptive filenames

## Saving Changes

### Save Options
- **Save**: Save as draft
- **Publish**: Make live immediately
- **Unpublish**: Remove from site

### Git Workflow
- Changes commit to Git
- Commit message auto-generated
- Can view in Git history
- Rollback available

## Preview

### Preview Before Publishing
1. Make edits
2. Click "Preview"
3. See how it looks
4. Make adjustments
5. Publish when ready

## Best Practices

1. **Preview First**: Always preview before publishing
2. **Save Often**: Don't lose work
3. **Use Alt Text**: For all images
4. **Check Links**: Verify all links work
5. **Test Mobile**: Preview on mobile view

## Troubleshooting

### Common Issues
- **Can't Save**: Check permissions
- **Images Not Showing**: Verify file paths
- **Formatting Lost**: Check markdown syntax
- **Changes Not Appearing**: May need cache clear

### Getting Help
- Check [Troubleshooting Guide](./troubleshooting.md)
- Contact administrator
- Check Git for errors
