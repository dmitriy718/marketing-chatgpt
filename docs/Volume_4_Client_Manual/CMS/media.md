# Media Management

## Uploading Media

### Images
1. Click media field
2. Choose "Upload"
3. Select image file
4. Add alt text
5. Save

### Files
1. Click file field
2. Choose "Upload"
3. Select file
4. Save

## Media Library

### Accessing Library
- Navigate to "Media" section
- View all uploaded files
- Search and filter
- Delete unused files

### Organizing Media
- Use descriptive filenames
- Group by content type
- Delete old/unused files
- Keep library organized

## Best Practices

### Image Optimization
- Compress before upload
- Use appropriate format (JPG for photos, PNG for graphics)
- Keep file sizes reasonable
- Add descriptive alt text

### File Management
- Regular cleanup
- Remove unused files
- Organize by project
- Use consistent naming

## Supported Formats

### Images
- JPG/JPEG
- PNG
- GIF
- WebP (if supported)

### Files
- PDF
- DOC/DOCX
- Other common formats

## File Size Limits

- **Images**: 10MB max
- **Files**: 25MB max
- **Recommendation**: Optimize before upload
